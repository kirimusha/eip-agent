# EIP Agent — Многоагентная система

Интеллектуальная платформа на базе **Qwen** с локальным запуском через Ollama.

## Доступные агенты

| Агент | Специализация | Инструменты |
|-------|--------------|-------------|
| 🤖 Универсальный | Общие задачи | Поиск, калькулятор |
| 🔍 Исследователь | Анализ и факты | Поиск, резюме |
| 💻 Программист | Код и алгоритмы | Калькулятор, поиск |
| 📚 Тьютор | Обучение | Поиск |

## Инструменты

- **web_search** — поиск в интернете через DuckDuckGo
- **calculator** — вычисление математических выражений
- **summarize_text** — сжатие длинных текстов

## Как пользоваться

1. Выберите агента кнопками в начале чата
2. Задайте вопрос — агент сам решит, нужны ли инструменты
3. В конце каждого ответа видно время генерации и модель

> Все данные обрабатываются локально. Интернет используется только для поиска.
