🔄 Поток данных

1️⃣ Пользователь пишет вопрос
→
2️⃣ Chainlit отправляет в app.py
→
3️⃣ @cl.on_message
4️⃣ AgentExecutor принимает решение
→
5️⃣ Вызов инструментов
→
6️⃣ LLM генерирует ответ
7️⃣ Форматирование ответа
→
8️⃣ Отправка в UI
→
9️⃣ Отображение пользователю

🛠️ Инструменты агента

🔍 Web Search

Описание: Поиск актуальной информации в интернете

Использование: Погода, новости, факты

API: DuckDuckGo Search Results

🧮 Calculator

Описание: Вычисление математических выражений

Использование: 2 + 2, 10 * 5, (3+4)*2

Механизм: eval() с безопасным окружением

📋 Пример цепочки вызовов

Пользователь: "Сколько будет 2 + 3567?" │ ▼ Агент: Thought → "Нужно вычислить математическое выражение" │ ▼ Агент: Action → calculator("2 + 3567") │ ▼ Инструмент: Observation → "3569" │ ▼ Агент: Thought → "Теперь могу ответить" │ ▼ Агент: Final Answer → "2 + 3567 = 3569" │ ▼ Пользователь получает: "2 + 3567 = 3569 (Время: 1.2s | Модель: Qwen 2.5 3B)"
